= Work examples of graphs using d3.js

Graphs
- Doughnut with rotation
- Time series with intersection slider
- Distribution with slider

To run examples:

in terminal run server, for example

python -m SimpleHTTPServer 3000

then in browser 

http://localhost:3000/graphs/doughnut

http://localhost:3000/graphs/time_series

http://localhost:3000/graphs/distributions
