= Work examples of graphs using d3.js

Graphs
- Doughnut with rotation
- Time series with intersection slider
- Distribution with slider

To see examples (required ruby):

1) (recomemded)

  rvm gemset use d3js-examples

2) run  

  ./dev.sh

or

  bundle
  ruby run.rb

3) then in browser 

http://localhost:9292/graphs/doughnut
http://localhost:9292/graphs/time_series
http://localhost:9292/graphs/distributions
